# sorting_benchmark
Testing time efficiency of six sorting algorithms:

- selection sort O(n^2)
- insertion sort O(n^2)
- heap sort O(nlogn)
- quicksort O(n^2)
- random quicksort O(nlogn) 
- shellsort O(n^2)
  
on five different number sequences:

- ascending
- descending
- constant
- A-shaped
- random
